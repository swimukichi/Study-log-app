Seedance 2.0 資料まとめ

EmilyさんのGitHub資料を、動画生成の初心者でも使いやすいように整理したページです。

結論：この資料は、Seedance 2.0で動画を作るための「プロンプトの書き方マニュアル」です。

動画生成ツール本体ではありません。Codexに読ませて、短い動画・連続動画・参照画像ありの動画を作るための指示文を整えるために使います。

これは何？ 基本ルール 参照素材 連続クリップ プロンプト型 使う流れ

1. これは何？

Seedance 2.0で動画を作るとき、AIに何をどう伝えるかを整理するための資料です。

できること

動画の内容、カメラ、光、動き、音、参照画像の使い方を整理できます。

できないこと

これだけで動画が無料生成できるわけではありません。Seedance 2.0を使えるサービスや画面は別に必要です。

2. 日本語プロンプトの基本

一番大事なのは、ふわっとした言葉を使いすぎないことです。

弱い書き方 映画みたいに エモく 雰囲気よく
強い書き方 固定カメラ 柔らかい窓光 ゆっくり振り返る 静かな環境音

つまり、「いい感じにして」ではなく、何が画面に起きるのかを分けて書きます。

  1. 誰・何を見せるか
  2. 何が動くか
  3. カメラはどう動くか、または動かさないか
  4. 光はどこから来るか
  5. 音は必要か
  6. 変えてはいけないものは何か

3. 画像・動画・音声を使うとき

[Image1]、[Video1]、[Audio1] のようなタグは翻訳せず、そのまま使います。

画像

人物の顔、商品の形、服装、背景、最初のフレーム、最後のフレームを固定するのに向いています。

動画

動き、カメラのリズム、歩き方、タイミングを参考にするのに向いています。

音声

テンポ、空気感、環境音、話し方の雰囲気を指定するのに向いています。

注意

1つの素材に何でもやらせないこと。素材ごとに役割を決めると安定します。

[Image1]は猫の見た目を固定する。
[Video1]はカメラの動きだけを参考にする。
[Audio1]はテンポだけを参考にする。
服装、部屋、ロゴ、人物の見た目は[Video1]からコピーしない。

4. 長い動画は1本で作らない

資料の考え方では、長い物語は「複数の短いクリップ」に分けます。

理由は、1回の生成に詰め込みすぎると、途中で内容がずれたり、最後が思った通りにならなかったりするからです。

  1. まず全体のゴールを決める
  2. Clip 01では1つの動きだけを見せる
  3. 生成された動画の最後を確認する
  4. 実際の終わり方に合わせてClip 02を書く
  5. これを繰り返す
大事：
予定では「猫がクッションになった」はずでも、
生成結果が「猫が丸まったところ」で終わったなら、
次のプロンプトは「丸まった猫」から始める。

5. 最初と最後のフレームを使う考え方

最初の画像と最後の画像を用意できる場合は、AIに「この2枚の間の動きだけを作って」と頼む考え方があります。

[Image1]は最初のフレーム。
[Image2]は最後のフレーム。
同じ猫の丸いシルエットと画面中央の位置を維持する。
猫がゆっくり沈み込み、柔らかいクッションの形へ連続的に変化する。
カメラは固定。光は柔らかい朝の窓光。
余計な文字、透かし、別の動物を追加しない。

変身動画では、最初と最後で残すものを決めると安定しやすいです。たとえば「丸いシルエット」「目線の位置」「同じ光」などです。

6. まず使うプロンプト型

最初はこの型だけ覚えれば十分です。

目的：
[何を見せたいか]

被写体：
[人物・猫・商品など]

動き：
[1つの動きだけ]

カメラ：
[固定 / ゆっくりドリーイン / 横移動 など]

光：
[柔らかい窓光 / 逆光 / 冷たい月明かり など]

音：
[静かな環境音 / 音楽なし / 小さな効果音 など]

変えてはいけないもの：
[顔、形、色、ロゴ、服装、背景など]

入れないもの：
字幕、透かし、余計な文字、別キャラクター

7. あなた向けの例

猫の5秒変身動画なら、こういう考え方が合います。

[Image1]の猫を参照し、顔の丸さ、毛色、目の雰囲気を維持する。
猫はベッドの上で丸くなり、体の輪郭がゆっくり柔らかいクッションへ変化する。
変化は小さく、かわいく、怖い要素は入れない。
カメラは固定の中景。柔らかい朝の窓光。
音は静かな部屋の環境音だけ。
字幕、透かし、余計な文字、別の動物を追加しない。

台湾神話 × バイオメカニカルなら、最初から詰め込みすぎず、1クリップごとに「仮面が開く」「金属の羽が少し動く」のように分けるのが安全です。

8. 権利まわりの注意

資料では、有名キャラクター、実在人物、ブランド、曲、声などをそのまま使うのは避ける考え方になっています。

使いたい雰囲気がある場合は、名前を隠して真似するのではなく、オリジナルの人物・世界観・衣装・光・カメラに置き換えます。

9. 実際に使う流れ

  1. 作りたい動画を一言で書く
  2. 5秒で見せる動きを1つだけ決める
  3. 参照画像や動画があるなら役割を決める
  4. Codexにプロンプトを作ってもらう
  5. Seedance 2.0に貼って生成する
  6. できた動画の最後を見て、次のクリップを作る
